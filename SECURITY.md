# Security Policy

## Supported Versions

We actively support and patch the latest release of `ghwm`.

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please open an issue using our **Security Report** template.

We will acknowledge your report within 48 hours and provide a timeline for addressing the issue. Thank you for helping keep this project secure!
